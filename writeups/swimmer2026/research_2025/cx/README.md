# cx (100pt / 523 solves)

## 問題文

2025年春、かつて香港に存在していた空港の100周年を記念して、特別なフライトが実施されたようです。  
このフライトの便名を解答してください。  
例えば、便名が `JL2026` のとき、Flag は `SWIMMER{JL2026}` となります。

In Spring 2025, it appears that a special flight was carried out to commemorate the 100th anniversary of the airport that once existed in Hong Kong.  
Answer the name of the flight number.  
If the flight was `JL2026`, the flag should be `SWIMMER{JL2026}`.


## 解法

`香港 空港 100周年` といった単語で Google 検索を行うと、以下のような記事がヒットします。

- [キャセイパシフィック航空、空港開発100周年で低空飛行　ビクトリアハーバーや旧啓徳空港周辺 - TRAICY](https://www.traicy.com/posts/20250329333965/)
- [「香港カーブ」が1日限りの復活！？ キャセイ、啓徳空港100周年で特別飛行実施](https://flyteam.jp/news/article/142976)

これより、旧啓徳空港（Kai Tak Airport）の100周年を記念し、キャセイパシフィック航空がCX8100便という特別フライトを実施したことがわかります。

Flag: **SWIMMER{CX8100}**
