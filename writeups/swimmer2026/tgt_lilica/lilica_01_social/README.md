# lilica_01_social (100pt / 483 solves)

## 問題文

`lilica` は2026年時点でクリエイターとして活動しており、「黄昏ブロッサムリリカ」と名乗っていたことが分かっています。
また、Xのアカウントを持っていたことが確認されています。

そのXアカウントのID（スクリーンネーム）を解答してください。

例えば、[@gov_online](https://x.com/gov_online) が対象のアカウントの場合、Flag は `SWIMMER{@gov_online}` となります。

In 2026, `lilica` was active as a creator and was known to use the name "黄昏ブロッサムリリカ".  
We have also confirmed that they had an X (Twitter) account.

Please answer the ID (screen name) of that X account.  
For example, if the target account were [@gov_online](https://x.com/gov_online), the flag would be `SWIMMER{@gov_online}`.


## 解法

調査指示書に記載されている lilica のハンドルネーム `黄昏ブロッサムリリカ` で検索すると以下のアカウントがヒットします。

[https://x.com/twilight_lilica](https://x.com/twilight_lilica)

Flag: **`SWIMMER{@twilight_lilica}`**
